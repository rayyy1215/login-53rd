Need Help?

  • Where can I get login assistance for Online Banking?

    Simply click on Trouble logging in?. Or, you can click on View All Help Topics, which appears on each screen.

  • What if I forgot my Online User ID?

    If you forget your Online User ID, simply click on Trouble logging in?. Then, select "I forgot my Online User ID". You will then be asked to enter information to identify yourself so your Online User ID can be provided to you.

  • How do I enable or disable the Online User ID save feature?

    Once you have entered your Online User ID, you can opt to save it by checking the "Save this Online User ID" box. When you save the Online User ID, only the last 4 characters will be visible within the Online User ID area on the Online Banking Login page.

    Saving of the Online User ID can be disabled by deleting the cookies within your browser.

  • How do I log into Online Banking if I'm a first-time user?

    Simply enter your Online User ID and click the 'Login' button, then answer your security question (if presented), and enter your Password. In some situations, your Online User ID will be your Debit Card number and your Password will be the last four digits of your Social Security number. If you haven't already selected an Online User ID, you will be asked to do so and to select a security Image and Phrase to log into Online Banking.
